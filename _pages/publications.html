  • FaaSBoard: Efficient Graph Processing with a Disaggregated Architecture on Serverless Services. Yushi Liu*, Yikang Ruan*, Letian Ruan, Zijun Li, Sen Gao, Weihao Cui, Shixuan Sun, Quan Chen, Shuo Quan, Jie Wu, Bingsheng He, and Minyi Guo. Paper to appear / Code, SIGMOD 2026.

    We introduce FaaSBoard, a graph processing system with a disaggregated serverless architecture for better computing performance and cost efficiency, equipped with image-based graph loading for faster data loading, proxy-based collective communication for better communication performance, 2D balanced graph partitioning for better load balance, and a proactive terminate-and-respawn mechanism for autonomous elasticity and cost savings. We develop FaaSBoard fully on AWS, using AWS Lambda, AWS Fargate, and other cloud infrastructure.

* Equal contribution.
